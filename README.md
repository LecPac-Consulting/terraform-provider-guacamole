# Terraform/OpenTofu Provider for Apache Guacamole

This provider allows you to manage [Apache Guacamole](https://guacamole.apache.org/) using Terraform/OpenTofu.
Available in [Terraform registry](https://registry.terraform.io/providers/LecPac-Consulting/guacamole/latest) and [OpenTofu registry](https://search.opentofu.org/provider/LecPac-Consulting/guacamole/latest).

This provider was highly inspired from [techBeck03/terraform-provider-guacamole](https://github.com/techBeck03/terraform-provider-guacamole).
